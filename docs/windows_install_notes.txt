Install OpenCV from their sourceforge page at https://sourceforge.net/projects/opencvlibrary/files/opencv-win/
(tested with opencv-4.1.2-vc14_vc15.exe)

Add main OpenCV folder to PATH (windows environment variable)
e.g. C:\Users\username\opencv\build

Tested with Visual Studio Community 2019
Installed packages (from installation manager):
	- desktop development (C++)
	- python (3.7.5) language support
	- python miniconda

Just use the Visual Studio opening screen, offering to clone a repository. Enter the URL after installing all necessary packages.

---------------------
Hardware requirements
---------------------
NVIDIA GPU with driver + CUDA installed

Create a new conda environment:

numpy tensorflow-gpu=1.13.1 matplotlib keras pillow opencv openssl
